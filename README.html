SWD Chat

一个免费，自由，开源的局域网去中心化信息传输交流工具

SWDChat v2.0.0rc1是由SWD Studio以GNU GPLv3协议于CST 2024年9月30日发布的软件

运行时请启动swdchat.py文件.

关于此应用的问题？

raise an issue or send an email to <swd_studio@outlook.com>
